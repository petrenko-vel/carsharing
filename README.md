# 🚗 CarSharing Application

Современное веб-приложение для сервиса каршеринга, построенное на React и TypeScript.

## 🛠 Стек технологий

- **Frontend**: React 19 + TypeScript
- **Сборка**: Vite
- **Управление состоянием**: Zustand
- **Работа с сетью**: Axios
- **Стилизация**: Sass (SCSS) + CSS Modules
- **Роутинг**: React Router 7
- **Качество кода**: ESLint (Airbnb config), Prettier, Stylelint

## 📂 Структура проекта

Проект организован по функциональному принципу для удобного масштабирования:

```text
src/
├── api/            # API-клиенты и запросы (Axios instance)
├── app/            # Глобальные настройки, провайдеры и точка входа (App.tsx)
├── components/     # Переиспользуемые компоненты
│   ├── common/     # Базовые атомарные элементы (Кнопки, инпуты)
│   ├── layout/     # Элементы структуры (Header, Footer, Sidebar)
│   ├── forms/      # Сложные формы (Фильтры, формы бронирования)
│   └── ui/         # Сложные UI-блоки (Карточки машин, слайдеры)
├── hooks/          # Кастомные React хуки (useAuth, useCars)
├── pages/          # Страницы приложения (Home, Booking)
├── styles/         # Глобальные стили, переменные и миксины
├── types/          # Общие TypeScript интерфейсы и типы
└── utils/          # Хелперы и утилиты (форматтеры, гео-расчеты)
```

## 🚀 Быстрый старт

1.  **Установка зависимостей**:

    ```bash
    npm install
    ```

2.  **Запуск в режиме разработки**:

    ```bash
    npm run dev
    ```

3.  **Сборка проекта**:
    ```bash
    npm run build
    ```

## 🧹 Проверка кода

- **Линтинг**: `npm run lint`
- **Форматирование**: `npm run prettier`
- **Проверка стилей**: `npm run stylelint`
